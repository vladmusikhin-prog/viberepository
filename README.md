# Telegram Bot Project

Базовая структура проекта для разработки Telegram-бота.

## Структура

- `Bot/` — код и конфигурация бота
- `Docs/` — спецификации и проектная документация

## Быстрый старт

1. Перейдите в папку `Bot/`.
2. Скопируйте `.env.example` в `.env`.
3. Заполните переменные окружения.
4. Реализуйте логику в `src/`.

## Рекомендации

- Ведите изменения требований в `Docs/REQUIREMENTS.md`.
- Фиксируйте архитектурные решения в `Docs/ARCHITECTURE.md`.
- Обновляйте план релизов в `Docs/ROADMAP.md`.
